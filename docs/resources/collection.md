# typesense_collection (Resource)

Manages a Typesense collection. Collections in Typesense are similar to tables in a database - they store documents that can be searched.

Each collection has a schema that defines the fields and their types. Typesense supports various field types including strings, numbers, booleans, arrays, geopoints, and nested objects.

## Example Usage

### Basic Collection

```terraform
resource "typesense_collection" "products" {
  name                  = "products"
  default_sorting_field = "popularity"

  field {
    name = "id"
    type = "string"
  }

  field {
    name  = "name"
    type  = "string"
    infix = true  # Enable infix search for partial matching
  }

  field {
    name  = "price"
    type  = "float"
    facet = true  # Enable faceting for filtering
  }

  field {
    name = "popularity"
    type = "int32"
    sort = true  # Enable sorting
  }
}
```

### Collection with Nested Objects

```terraform
resource "typesense_collection" "orders" {
  name                 = "orders"
  enable_nested_fields = true

  field {
    name = "id"
    type = "string"
  }

  field {
    name = "customer"
    type = "object"
  }

  field {
    name = "items"
    type = "object[]"
  }

  field {
    name = "total"
    type = "float"
    sort = true
  }
}
```

### Collection with Custom Tokenization

```terraform
resource "typesense_collection" "documents" {
  name             = "documents"
  token_separators = ["-", "_", "."]
  symbols_to_index = ["#", "@"]

  field {
    name  = "title"
    type  = "string"
    infix = true
  }

  field {
    name = "content"
    type = "string"
  }

  field {
    name     = "tags"
    type     = "string[]"
    facet    = true
    optional = true
  }

  field {
    name   = "body"
    type   = "string"
    locale = "en"  # Language-specific processing
  }
}
```

## Field Types

Typesense supports the following field types:

| Type | Description |
|------|-------------|
| `string` | Text field for full-text search |
| `string[]` | Array of strings |
| `int32` | 32-bit signed integer |
| `int64` | 64-bit signed integer |
| `float` | Floating-point number |
| `bool` | Boolean value |
| `geopoint` | Geographic coordinates (lat, lng) |
| `geopoint[]` | Array of geographic coordinates |
| `object` | Nested object (requires `enable_nested_fields = true`) |
| `object[]` | Array of nested objects |
| `auto` | Automatic type detection |
| `string*` | Auto-detect string or string[] |

## Import

Collections can be imported using the collection name:

```shell
terraform import typesense_collection.products products
```
