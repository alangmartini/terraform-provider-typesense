# typesense_stopwords_set (Resource)

Manages a Typesense stopwords set. Stopwords are common words (like "the", "a", "an") that are filtered out from search queries to improve search relevance and performance.

Stopwords sets can be applied globally or to specific collections during search queries.

## Example Usage

### English Stopwords

```terraform
resource "typesense_stopwords_set" "english" {
  name   = "english-stopwords"
  locale = "en"
  stopwords = [
    "the", "a", "an", "and", "or", "but", "in", "on", "at", "to", "for",
    "of", "with", "by", "from", "as", "is", "was", "are", "were", "been",
    "be", "have", "has", "had", "do", "does", "did", "will", "would",
    "could", "should", "may", "might", "must", "shall", "can", "need",
    "it", "its", "this", "that", "these", "those"
  ]
}
```

## Using Stopwords in Search

When searching, you can specify which stopwords set to use:

```bash
curl "https://your-cluster.typesense.net/collections/products/documents/search" \
  -H "X-TYPESENSE-API-KEY: ${TYPESENSE_API_KEY}" \
  -d '{
    "q": "the best running shoes",
    "query_by": "name,description",
    "stopwords": "english-stopwords"
  }'
```

With stopwords applied, the query "the best running shoes" would effectively search for "best running shoes".

## Import

Stopwords sets can be imported using the stopwords set name:

```shell
terraform import typesense_stopwords_set.english english-stopwords
```
